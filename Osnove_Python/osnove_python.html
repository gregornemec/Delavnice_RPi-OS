Osnove programiranja v programskem jeziku Python

V tem dokumentu bodo osnovni in poučni primeri uporabe programskega jezika Python. Primere programske kode prepiši ali jih kopiraj in jih preizkusi.

1 Dober prvi program

Programski jezik nam omogoča pisanje navodil za računalnik. Navadno računalniku damo kopico podatkov in z navodili (programom), želimo da te podatke obdela in jih izpiše. Izpis podatkov je torej sestavni del večine programov. Danes bomo spoznali kako podatke izpišemo na zaslon.

Programski jezik uporablja rezervirane besede, ki jih uporabljamo kot ukaze. Ukaz za izpis na zaslon je print temu sledita () v katerem izpišemo željeno vsebino. Za izpis teksta moramo tega dati v "Tekst", temu pravimo, da smo izpisali niz znakov.

1.1 Program 01 - Pozdravljen svet

Vsak ukaz print ("Tekst.") izpiše vsebino v novo vrstico.

Naloga: Prilagodi izpise Besedila po svoje. 

print ('Pozdravljen svet!')
print ('Pozdravljen še enkrat!')
print ('Zapiši svoj tekst')
print ('Nova vrstica z novim besedilom.')

2 Spremenljivke

Spremenljivke uporabljamo za to ta shranjujemo različne vrste vrednosti. Vrednosti so lahko različne, lahko je to tekst, celo število ali kaj kompleksnega kot je seznam števil. Spremenljivko ustvarimo oz. jo deklariramo tako, da ji določimo ime in ji dodamo zank je enako, spremenljivka =. Na drugi strani zapišemo vrednost, ki jo želimo shraniti. Pri spremenljivkah moramo paziti na to, da ne zberemo rezervirano ime kot je na primer print

Naloga: Ustvari spremenljivko ime in ji priredi vrednost svojega imena. Spremenljivko tudi izpiši. V isto vrstico dodaj še tisto kar izpisuješ 'ime: '

4 Pogoji in odločanje

Računalnik nam poleg računanja omogoča tudi odločanje v odvisnosti od pogojev če so ti resnični ali niso resnični. Za odločanje uporabljamo pogojni stavek if else. Za lažje razumevanje si poglejmo primer.

Pogojni uporabljajo naslednje osnovne matematične operatorje.

operator opis
== je enako
< manjše
<= manjše kvečjemu enako
> večje
>= večje kvečjemu enkao

Naloga: Prepiši (kopiraj) program in spreminjaj spremenljivki a in b. Kako to vpliva na izpis programa.

4.1 Program 04 - Pogojni stavek

a = 10
b = 7

if a < b:
    print ('a je manjsi od b')
else:
    print ('a je vecji od b')
